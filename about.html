About TextScribe OCR

TextScribe is an advanced, 100% static OCR engine originally conceived as a deep learning backend, now optimized for the browser using Tesseract.js to ensure complete data privacy.

Project Overview

TextScribe implements a complete offline handwriting recognition system using Tesseract.js. By operating entirely on the client side, it eliminates the need for RESTful endpoints or backend servers, providing highly accurate, secure text extraction straight from a responsive web dashboard.

Originally conceived as an AI deep learning backend using PaddleOCR and FastAPI, the system has been refactored to run directly in the browser via WebAssembly. This ensures that sensitive documents are never uploaded to remote servers.

Key Features

  • Client-side offline functionality
  • Privacy-first data processing
  • Drag-and-drop web interface
  • Automatic progress streaming
  • Export directly to .txt
