About Project

Handwriting Recognition System

An offline handwriting text recognition system built using deep learning for academic purposes. Convert handwritten documents to digital text with complete privacy.

Project Overview

This project implements a complete offline handwriting recognition system using Convolutional Recurrent Neural Networks (CRNN). The system can convert images of handwritten text into editable digital text without requiring any internet connection.

The solution is designed as an academic project to demonstrate the practical application of deep learning in optical character recognition (OCR). It combines computer vision techniques with sequence modeling to achieve accurate text recognition.

Key Features

  • Complete offline functionality
  • Deep learning-based recognition
  • Automatic image preprocessing
  • Web-based user interface
  • Export to text files

CNN Feature Extractor

  • 7 Convolutional layers with BatchNorm
  • ReLU activation functions
  • MaxPooling for downsampling
  • Dropout for regularization
  • Output: 512 feature channels

BiLSTM Sequence Modeler

  • 2-layer bidirectional LSTM
  • 256 hidden units per direction
  • Captures sequential dependencies
  • Layer normalization for stability
  • Output: 512-dimensional features

CTC Loss & Decoding

  • Connectionist Temporal Classification
  • No character alignment needed
  • Handles variable-length sequences
  • Greedy decoding for inference
  • Blank token for repeated characters

Structure

Project Structure

handwriting-recognition/
├── config.py               # Centralized configuration flags for the entire pipeline
├── main.py                 # Runnable CLI example script
├── app.py                  # FastAPI application — defines REST API routes
├── pipeline.py             # Core orchestrator: routes data through preprocessing → inference
├── requirements.txt        # Pinned dependency versions
│
├── preprocessing/          # Modular computer vision preprocessing steps
│   ├── resize.py           # Resize to fixed height with aspect ratio preservation
│   ├── grayscale.py        # RGB to grayscale conversion
│   ├── denoise.py          # Non-local means denoising
│   ├── clahe.py            # Contrast Limited Adaptive Histogram Equalization
│   ├── threshold.py        # Otsu / adaptive binarization
│   ├── morphology.py       # Morphological operations (open, close, dilate, erode)
│   └── deskew.py           # Skew detection and correction
│
├── model/                  # Deep learning model components
│   ├── cnn_backbone.py     # 7-block CNN for visual feature extraction
│   ├── bilstm.py           # Bidirectional LSTM for sequence modeling
│   ├── dense.py            # Dense layer mapping features to character classes
│   └── crnn.py             # Unified CRNN model combining CNN + BiLSTM + Dense
│
├── ctc/                    # CTC (Connectionist Temporal Classification)
│   ├── ctc_loss.py         # CTC loss for training
│   └── ctc_decoder.py      # Greedy CTC decoder for inference
│
├── utils/                  # Utility helpers

|    └── visualizer.py       # Debug visualization tools
│
├── static/                     # Static assets
